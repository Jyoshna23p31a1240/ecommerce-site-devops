☁️ E-Commerce Website Deployment using AWS DevOps

This project demonstrates the deployment of an e-commerce web application on AWS using DevOps practices. The primary objective was to automate the deployment process through a Continuous Integration and Continuous Deployment (CI/CD) pipeline, enabling seamless updates whenever new code is pushed to GitHub.

The project was completed as part of an AWS DevOps Internship, providing practical experience with cloud deployment and modern DevOps workflows.

🚀 Project Objectives
Deploy a web application on AWS Cloud
Implement an automated CI/CD pipeline
Integrate GitHub with AWS services
Automate application deployment after every code push
Learn industry-standard DevOps practices

⚙️ Workflow
Developer pushes code to GitHub.
AWS CodePipeline detects the changes.
CodeDeploy deploys the latest version to the EC2 instance.
The updated website becomes available automatically without manual deployment.
